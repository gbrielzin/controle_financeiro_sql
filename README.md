# Controle Financeiro com SQL

## Objetivo do Projeto
Este projeto foi desenvolvido com o objetivo de praticar SQL em um cenário real, utilizando dados pessoais de despesas mensais para organizar informações financeiras e extrair insights úteis para tomada de decisão.

## Contexto
Morando sozinho, identifiquei a necessidade de entender melhor meus gastos mensais. Para isso, criei uma base de dados no PostgreSQL com despesas organizadas por categorias, transformando uma necessidade pessoal em um projeto prático de análise de dados.

## Ferramentas Utilizadas
- PostgreSQL
- SQL
- GitHub

## Estrutura dos Dados
A base foi organizada em categorias de gastos, como:
- Aluguel
- Mercado
- Transporte
- Internet
- Outras despesas recorrentes

## Análises Realizadas
Durante o projeto, foram aplicadas consultas SQL para:

- Selecionar e visualizar dados financeiros
- Agrupar despesas por categoria com `GROUP BY`
- Identificar os maiores gastos mensais
- Comparar padrões de consumo
- Transformar dados brutos em informações úteis

## Principais Aprendizados
- Criação de tabelas estruturadas no PostgreSQL
- Inserção de dados históricos para análise realista
- Uso de consultas SQL para análise exploratória
- Aplicação de agrupamentos e lógica analítica
- Interpretação de dados para gerar insights financeiros

## Próximos Passos
Como evolução do projeto, o próximo passo será integrar esses dados ao Power BI para construir um dashboard interativo, permitindo visualizar:

- Gastos por categoria
- Evolução mensal das despesas
- Indicadores financeiros
- Insights visuais para decisões mais estratégicas

## Estrutura do Projeto
- `controle_financeiro.sql` → criação da tabela, inserção de dados e consultas SQL
- `README.md` → documentação do projeto

## Conclusão
Este projeto demonstra como SQL pode ser utilizado para transformar dados financeiros em insights práticos, unindo organização, análise e visão estratégica. Além de fortalecer minhas habilidades técnicas, também reforça minha evolução como profissional em formação na área de Análise de Dados.
